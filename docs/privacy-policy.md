# 交我选隐私政策

生效日期：2026-07-18

“交我选”是一个在 Windows 本机运行的上海交通大学课程监控与选课方案工具，并非上海交通大学官方产品。

## 本应用处理的数据

- 用户主动输入的 JAccount、选课社区账号密码，以及可选的 SMTP 邮箱配置；
- 教务系统返回的姓名、学号、专业、已选课程、课程目录、容量和评分缓存；
- 用户创建的课程方案、监控状态、换课记录和应用日志。

## 使用、存储与传输

- 凭据使用 Windows DPAPI 安全存储；密码不会在界面回显，也不会随应用包或日志上传；
- 课程相关数据、监控状态和缓存保存在本机应用数据目录，仅用于课程展示、监控、通知和换课功能；
- 仅在用户明确保存账号、同步课程、刷新评分、启动监控或发送测试邮件时，应用才会向对应的教务、课程社区或用户配置的 SMTP 服务发起请求；
- 本应用不提供开发者运营的遥测、广告或用户数据汇集服务，不会将上述数据出售或共享给第三方。

## 用户控制与删除

用户可以停止监控、关闭邮件通知并更新账号配置。若要移除本机保存的数据与凭据，请在 Windows 设置中卸载“交我选”并选择删除应用数据；重新安装后需要重新完成首次设置。

## 外部服务与联系方式

使用 JAccount、教务系统或选课社区前，请确保你有权使用相应服务，并遵守其适用规则。本政策更新或隐私问题请通过项目主页提交 issue：<https://github.com/tangmubai/sjtu-monitor/issues>。
